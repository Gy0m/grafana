# Divider

Used to create separation between large sections of page content. Or used to separate smaller sections
of components. i.e in the info section of the header.

### Usage

```tsx
import { Divider } from '@grafana/ui';

   <header>
        <h1>My title here</h1>
        <img src="logo.png" alt="logo" />
    </header>
    <Divider />
    <main>
        <p>Main content goes here</p>
    </main>


```
